# 在线计算工具站 — 需求文档（V1.0）

## 1. 项目概述
### 1.1 项目名称（暂定）
- 中文名：在线实用计算工具站
- 英文名（可选）：EasyCalc / SmartCalc / CalcTools
- ⚠️ 正式名称后期可改，不影响结构

### 1.2 项目目标
- 构建一个 长期可维护的在线计算工具网站
- 覆盖 工资 / 日期 / 学生 / 程序员 / 生活计算 等高频需求
- 支持 移动端优先
- 满足 Google AdSense 合规要求
- 不依赖后端， 纯前端即可运行

### 1.3 目标用户
- 上班族
- 学生
- 程序员
- 日常生活计算需求用户

## 2. 网站整体结构（非常重要）
### 2.1 URL 结构（SEO 友好）
- `/`                      首页
- `/salary/`               工资与收入计算
- `/date/`                 日期时间计算
- `/student/`              学生与考试工具
- `/dev/`                  程序员工具
- `/life/`                 生活常用计算
- `/about`                 关于我们
- `/privacy`               隐私政策（AdSense 必须）
- `/terms`                 使用条款
- `/contact`               联系我们

### 2.2 工具页面 URL 规范
- `/salary/tax-calculator`
- `/salary/bonus-tax`
- `/date/workday-calculator`
- `/student/gpa-calculator`
- `/dev/timestamp-converter`
- `/life/bmi-calculator`

👉 一个工具 = 一个独立页面

## 3. 网站布局设计（统一模板）
### 3.1 全站布局结构
**[ Header ]**
- Logo
- 主导航（6 大分类）

**[ Main Content ]**
- 页面标题
- 工具表单区
- 计算结果区
- 解释说明区
- 常见问题（FAQ）

**[ Sidebar / Inline Ads ]**
- 广告位（可选）

**[ Footer ]**
- 关于我们
- 隐私政策
- 使用条款
- 联系方式
- © 年份

### 3.2 Header（顶部）
- Logo（文字即可）
- 导航菜单：
  - 工资计算
  - 日期时间
  - 学生工具
  - 程序员工具
  - 生活计算
- 👉 移动端：汉堡菜单

### 3.3 工具页面布局（核心）
- H1：工具名称（如：税后工资计算器）
- **[ 工具输入区 ]**
  - 输入框
  - 下拉选项
  - 计算按钮（或实时计算）
- **[ 计算结果区 ]**
  - 核心结果（大号字体）
  - 辅助数据
- **[ 解释说明区 ]**
  - 结果含义
  - 使用建议
  - 注意事项
- **[ 常见问题 FAQ ]**
  - 2 ~ 5 个问题
- 👉 解释说明 + FAQ 是 SEO 和 AdSense 通过率关键

## 5. 广告（Google AdSense）需求
### 5.1 广告原则
- 不影响工具正常使用
- 不诱导点击
- 不遮挡内容

### 5.2 推荐广告位
- 工具下方（结果区后）
- 页面底部
- 首页列表中间

### 5.3 禁止行为
- ❌ 自动刷新广告
- ❌ 点击诱导
- ❌ 成人 / 博彩内容

## 8. 技术需求
### 8.1 技术栈
- HTML5
- CSS3（响应式）
- 原生 JavaScript (Note: Project will be initialized with React-TS for maintainability but output static assets)

### 8.2 性能要求
- 页面加载 < 2 秒
- 无阻塞脚本
- 支持手机浏览器

### 8.3 数据处理
- 所有计算逻辑前端完成
- 可使用 localStorage 保存输入

## 9. 合规与通过 AdSense 的关键点
- ✅ 独立页面
- ✅ 原创内容
- ✅ 有隐私政策 / 使用条款
- ✅ 有关于我们
- ✅ 有联系页面
- ✅ 非空站（至少 10+ 页面）
